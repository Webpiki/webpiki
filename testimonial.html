Step 1: Link required files

Hi Folks ! First and most important, the jQuery library needs to be included (no need to download - link directly from Google). Next, download the package from this site and link the Testimo CSS file (for the theme) and the Testimo Javascript file. we are also thankfull for bx slider & bootstrap.


<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
<script type="text/javascript" src="js/testimo.js"></script>
<link rel="stylesheet" href="css/demo.css" />

Step 3: Call the bxSlider

Call .bxslider() on <div class=".testimonials-slider">. Note that the call must be made inside of a $(document).ready() call, or the plugin will not work!


<script>
     jQuery('.testimonials-slider').bxSlider({
			slideWidth: 600,
			minSlides: 1,
			maxSlides: 1,
			slideMargin: 32,
			auto: true,
			autoControls: false
		  });
</script>
